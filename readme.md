## 绘制网络

### coin rate
coin rate的结果绘制为网络图，包括了挑选：
1. cr>0.5的作为连边的两种类型的图
2. cr最大的对象的连边
3. cr>0.5的作为连边且保留每个节点最大的边

一共4张图  

### SD 数据
来自文献的sd数据：
1. 挑选收信比例>50%的对象，连边
2. 挑选收信比例为所有对象的最大值的连边
